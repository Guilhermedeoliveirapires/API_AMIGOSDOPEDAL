# Pasta de Imagens das Rotas

## Estrutura de Arquivos

Coloque as imagens das rotas nesta pasta com os seguintes nomes:

- `rota-1.jpg` - Primeira rota (Represa de Embu-Guaçu)
- `rota-2.jpg` - Segunda rota (Trilha da Mata)
- `rota-3.jpg` - Terceira rota (Circuito 100 Rumo)

## Dicas

- **Formato**: Use JPG para fotos
- **Tamanho**: Otimize as imagens para web (recomendado: menos de 500KB por imagem)
- **Dimensões**: Qualquer proporção (será ajustada automaticamente para 200px de altura)
- **Conteúdo**: Fotos das rotas, paisagens, ou momentos dos pedais nessas rotas

## Observação

Se você tiver mais de 3 rotas, pode adicionar mais cards no HTML e mais imagens aqui. Basta seguir o padrão `rota-4.jpg`, `rota-5.jpg`, etc.
