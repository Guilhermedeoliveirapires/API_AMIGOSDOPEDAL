# Pasta de Assets

## Estrutura de Arquivos

Coloque as imagens nas seguintes pastas:

### Logo
- `logo.png` - Logo do grupo (formato PNG com fundo transparente recomendado)

### Hero Section (Home)
- `hero-bg.jpg` - Imagem de fundo para a seção inicial (recomendado: 1920x1080px ou maior)

### Galeria
- `galeria/galeria-1.jpg` - Primeira foto da galeria
- `galeria/galeria-2.jpg` - Segunda foto da galeria
- `galeria/galeria-3.jpg` - Terceira foto da galeria
- `galeria/galeria-4.jpg` - Quarta foto da galeria
- `galeria/galeria-5.jpg` - Quinta foto da galeria
- `galeria/galeria-6.jpg` - Sexta foto da galeria
- `galeria/galeria-7.jpg` - Sétima foto da galeria
- `galeria/galeria-8.jpg` - Oitava foto da galeria

## Dicas

- **Formato**: Use JPG para fotos e PNG para logo (com transparência)
- **Tamanho**: Otimize as imagens para web (recomendado: menos de 500KB por imagem)
- **Dimensões**: 
  - Logo: 200x200px ou maior (será redimensionado automaticamente)
  - Hero: 1920x1080px ou maior
  - Galeria: Qualquer proporção (será ajustada automaticamente)

## Nomes dos Arquivos

Se você tiver imagens com nomes diferentes, renomeie-as conforme a estrutura acima ou atualize os caminhos no arquivo `index.html`.
